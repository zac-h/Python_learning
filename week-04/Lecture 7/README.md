# Exercises
- Validate user's input IP address, returning invalid if the format is incorrect or any part of the IP number exceeds 255.
- Extract the location from the tail of a youtube url address, and return the address in shortened youtu.be format.
- Capture time input in 12 hour format and return time in 24 hour format. Handling multiple formats such as 9 AM, 9:00 AM.
- Count the number of a specific set of string characters in user input and return count. Differentiate between standalone characters "um" and characters in word.
- Import regular expression library (validator-collection) and use to validate email input by user, returning valid or invalid
